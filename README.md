# 📊 Job Market Analytics Dashboard (Power BI)

## 📌 Project Overview

This project analyzes global data-related job salaries using **Power BI**. The dataset was cleaned in **Google Sheets**, analyzed with **SQL**, and transformed into an interactive dashboard to uncover salary trends and job market insights.

---

## 🛠️ Tech Stack

* Power BI
* Google Sheets
* SQL

---

## 📊 Dashboard Features

* 💰 Global Average Salary KPI
* 📈 Salary Trend (2020–2022)
* 🏆 Highest Paying Job Roles
* 👨‍💼 Salary by Experience Level
* 🎯 Interactive Job Title Filter

---

## 📈 Key Insights

* Data Analytics Lead has the highest average salary.
* Executive-level professionals earn the highest salaries.
* Salary trends changed significantly between 2020 and 2022.
* Interactive filters enable easy comparison across job roles.
